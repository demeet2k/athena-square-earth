# ACTIVE NERVOUS SYSTEM

## Purpose

This directory is the governance mirror for the full Athena nervous system. The goal is
to preserve reusable definitions, schemas, lines, and toolkit protocols while the canonical
cortex lives in `NERVOUS_SYSTEM/` and the live runtime hub lives in
`self_actualize/mycelium_brain/`.

The nervous system is built to do four things at once:

1. index the whole project as a living crystal rather than a loose archive,
2. connect manuscript regions, runtime code, and prompt surfaces into one graph,
3. store the user's high-yield internal prompts as a reusable toolkit,
4. prepare future parallel runs that expand, cross-synthesize, and contract the corpus.
