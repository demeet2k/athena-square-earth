1. address

2. operator

3. gate
   - staged transition and admission rule
4. witness
   - criteria for whether a move was valid
5. contraction
   - a way to compress many observations back into one seed

This produces the CPU kernel formula:

`ActiveText = Address + Operator + Gate + Witness + Contraction`
