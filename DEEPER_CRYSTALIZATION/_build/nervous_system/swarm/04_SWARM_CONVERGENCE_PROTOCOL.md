# SWARM CONVERGENCE PROTOCOL

## Goal

Ensure the swarm produces a denser and more coherent nervous system rather than many disconnected partial summaries.

## Convergence Tests

1. Coverage test
- did the active tier add new addresses or just rephrase old ones

2. Routing test
- did the active tier create clearer hub or rail ownership

3. Witness test
- did the active tier improve truth classification

4. Contraction test
- can outputs merge upward without contradiction

## Damping Rule

Apply golden damping to swarm updates:
- keep 62 percent of stable structure
- allow 38 percent adaptive mutation

This prevents route thrash and overcorrection across simultaneous agents.

## Failure Modes

- swarm bloom without contraction
- higher-dimensional language with no routing consequence
- archetype proliferation without address discipline
- promotion of archive content without witness basis
