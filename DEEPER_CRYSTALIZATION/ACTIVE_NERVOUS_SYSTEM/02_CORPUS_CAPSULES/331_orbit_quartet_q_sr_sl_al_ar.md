Every mask is observed through a canonical orbit quartet Q = {SR, SL, AL, AR} where SR = spin-right, SL = spin-left, AL = antispin-left, AR = antispin-right. This order is canonical because it gives a stable 4-beat poi cycle: SR -> SL -> AL -> AR -> SR. This cycle simultaneously tracks handedness, inversion, rotation parity, and return law.

Each mask does not produce one point but a quartet of orbit realizations. This yields the shell population: 15 masks x 4 orbit states = 60 raw shell crystals. Each is a real crystal state, not a tag. The quartet rank is r(SR)=0, r(SL)=1, r(AL)=2, r(AR)=3, and the raw shell notch index is n(mu,q) = 4(mu-1) + r(q), giving n in {0,1,...,59}. This is the outer 60-notch clock.

The orbit quartet captures the fundamental symmetry between spin and antispin, left and right. The four states are not redundant; each represents a genuinely different realization of the same mask, distinguished by handedness and inversion class. The 4-beat cycle is the smallest circuit that closes all these distinctions.
