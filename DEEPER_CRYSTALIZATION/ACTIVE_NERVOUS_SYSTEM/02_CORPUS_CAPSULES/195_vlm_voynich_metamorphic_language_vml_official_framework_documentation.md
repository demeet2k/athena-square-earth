INTRODUCTION 1.1 Purpose of the VML Framework The Voynich Metamorphic Language (VML) interprets MS 408 not as an encrypted tongue but as a functional code of transformation. Within this view, each cluster of glyphs is a procedural unit—a compact instruction that governs the alteration of substance, energy, or state. Rather than transcribing speech, the manuscript encodes operations: heating, fixing, infusing, circulating, or refining. This framework provides the tools necessary to read those operations with precision. The purpose of this document is to establish a complete, self-contained grammar capable of translating every Voynich token into its operative meaning.
