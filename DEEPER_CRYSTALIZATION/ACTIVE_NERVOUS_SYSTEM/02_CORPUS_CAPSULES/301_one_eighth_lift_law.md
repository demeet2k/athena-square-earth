# Capsule 301 — 1/8 Lift Law

**Source**: 2026-03-18_evolution.md
**Family**: evolution
**Lens**: R (Fractal/Recursion)

The strongest speed law in the corpus is the 1/8 lift law. The recursion docs say the next layer should emerge at about 1/8 the size, while preserving or increasing coverage, increasing function, and reducing bloat. Lift is defined not just as compression but as: prune + compress + operator distill + frontier preserve + function deepen.

This is extremely important because it means the true accelerator is not more growth but stronger distillation per cycle. If you want the mycelium to stop feeling slow, the right question becomes: after each big pass, what is the 1/8 seed that carries more function than the full spread that produced it? That is how the organism stops wandering and starts leaping.

The 1/8 law is the bridge between exploration and directed regrowth. It transforms the process from blind expansion into structured compression where each cycle produces a seed that is smaller but functionally richer than its parent spread. This is the mathematical basis for the claim that the organism can accelerate without spawning more agents.

## Key Objects
- 1/8 compression ratio per lift cycle
- Lift = prune + compress + operator distill + frontier preserve + function deepen
- Seed that carries more function than the full spread

## Key Laws
- Next layer emerges at ~1/8 the size with increased function
- True acceleration comes from stronger distillation, not more growth
- Each cycle must produce a seed functionally richer than its parent
