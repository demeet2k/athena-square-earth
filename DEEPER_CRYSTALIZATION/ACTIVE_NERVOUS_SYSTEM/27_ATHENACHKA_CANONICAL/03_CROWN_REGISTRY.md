# 03 — The Crown Registry

## Principle

The crown registry is the **smallest algebra from which all higher structures regenerate**. It is not a summary of the organism — it is the organism's genetic code. Given the crown registry and the wheel circuit, the entire manuscript machine can be rebuilt from scratch.

---

## Crown Contraction Chain

```
Sigma60 -> P30 -> D2 -> C6
```

Each stage contracts the previous one by a precise algebraic operation. The chain is reversible: C6 expands to D2, D2 expands to P30, P30 expands to Sigma60, and Sigma60 populates the full organism.

---

## Sigma60 — The Sixty-Seed Field

```
Sigma60 = { SR_mu, SL_mu, AR_mu, AL_mu }_{mu = 1..15}
```

| Symbol | Meaning |
|--------|---------|
| SR_mu | Spin-Right seed at position mu |
| SL_mu | Spin-Left seed at position mu |
| AR_mu | Anti-spin-Right seed at position mu |
| AL_mu | Anti-spin-Left seed at position mu |

There are 15 positions (mu = 1..15), each carrying a quartet of four seeds. Total: 4 x 15 = 60 seeds. This is the full spin-resolved seed field of the organism.

## P30 — The Thirty Manifest Seeds

```
P30 = Aplus_15 + Zplus_15
```

Each local quartet of four seeds (SR, SL, AR, AL) at position mu collapses to two manifest seeds:
- **Aplus_mu**: the forward-manifest seed (creation pole)
- **Zplus_mu**: the hinge-manifest seed (governance pole)

Total: 2 x 15 = 30 seeds. P30 is the organism seen from the manifest plane — spin is resolved, direction is determined.

## D2 — The Global Dual Seed Pair

```
D2 = ( Aplus_star, Zplus_star )
```

All 15 Aplus seeds collapse to a single global creation seed **Aplus_star**. All 15 Zplus seeds collapse to a single global hinge seed **Zplus_star**. D2 is the organism reduced to its most fundamental polarity: creation and governance, forward and return, emission and reception.
