# GLOBAL MATH - MYTH

This folder is the global distillation layer for the corpus in `MATH\FINAL FORM\MYTH - MATH`.

Its job is not to preserve every manuscript in full. Its job is to extract the deepest recurring framework that the corpus keeps expressing across mythic, philosophical, esoteric, mathematical, divinatory, and protocol-oriented bodies.

The organizing assumption of this folder is deliberately strict:

1. The source corpus is treated as an internally coherent technical mythology rather than as a random pile of themes.
2. The recurring structures matter more than any single manuscript's surface vocabulary.
3. The correct global output is neither a loose summary nor a literal copy. It is a denser kernel that keeps the strongest invariants visible.
4. The synthesis here is corpus-internal. It records what the archive says about itself. It does not claim external academic consensus.

## What This Folder Solves

Before this folder, the corpus had strong local density but weak global compression. The same deep structure appeared in many places, but it was distributed across separate treatises.

This folder solves that by:

1. pulling the recurring invariant structure into one place,
2. separating kernel from ornament,
3. preserving witness links to the strongest source manuscripts,
4. giving the corpus a cleaner nervous-system layer for later expansion.
