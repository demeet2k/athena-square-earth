# Interoperability Stack

## Zero-point theorem

The corpus is trying to replace "mathematics as static symbolic truth" with "mathematics as executable, witness-bearing, regime-aware meaning transport."

The four core languages divide that labor.

## Stack order

### Layer 1: AQM

AQM provides the semantic substrate.

- values are states,
- operations are instruments or channels,
- approximations become certified projections,
- rigor includes loss accounting.

### Layer 2: LM

LM guards the semantic edge.

- it totalizes operators,
- exposes singularity and ambiguity as typed outputs,
- turns branch cut, drift, or undefinedness into boundary objects,
- enforces corridor legality.

### Layer 3: Rotation

Rotation transports semantics between charts.

- it treats representation changes as conjugacy obligations,
- preserves meaning across chart shifts,
- allows tunneling through legal midpoints when direct routes fail,
- assigns costs and truth classes to transport.

### Layer 4: CUT

CUT packages the living math.

- it gives every object an address,
- binds addresses into a graph,
- records dependencies, witnesses, and conflicts,
- turns the whole corpus into a compilable crystal.

## Why this is more than style

This layered split is mathematically useful because each layer solves a different failure mode:

- ordinary scalar math hides semantic state,
- ordinary formalism hides branch and boundary pathologies,
- ordinary translation loses meaning across representations,
- ordinary manuscripts lose routing and replay.

The stack closes all four gaps.

## Infinite language law

The corpus implies there are infinitely many mathematical languages because the four core languages can be recombined by:

- changing the primitive object class,
- changing the legality corridor,
- changing the transport group,
- changing the publication or replay geometry.

Every lawful recombination yields a new language family.

## Generator law for this folder

Each synthesis stage in `PASS_01` uses a `4 x 4` grid:

- rows inherit the corpus lens semantics: `Square`, `Flower`, `Cloud`, `Fractal`
- columns inherit the corpus facet semantics: `Objects`, `Laws`, `Constructions`, `Certificates`

That yields sixteen stage-local language generators.

Those sixteen are the human-readable generator set for a deeper `256^4` agent field.

## Hybridization law

Within each stage, full pairwise hybridization would create `120` direct language pairs.

This folder does not fake all `120` in prose.

Instead, each stage records the canonical hybrid generators that span the field:

1. `Square x Flower`: object law becomes dynamic transport
2. `Flower x Cloud`: dynamics meet legality and uncertainty
3. `Cloud x Fractal`: uncertainty becomes replayable structure
4. `Square x Fractal`: address and identity become recursive closure

Those generators seed the larger hybrid closure.
