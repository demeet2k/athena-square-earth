- its own primitives,
- its own legality rules,
- its own object types,
- its own operator semantics,
- its own certificate discipline,
- and its own way of composing or transporting meaning.

The first pass in this folder therefore does not flatten them into one language. It treats them as a stack, then uses the stack to generate new languages.
