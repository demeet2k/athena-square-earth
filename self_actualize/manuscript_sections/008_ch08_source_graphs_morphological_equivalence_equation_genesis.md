## 8.1. Square - Source Graphs and Morphological Objects

### 8.1.1. Source object and source graph

A source object is any bounded origin-bearing artifact or artifact slice from which lawful graph structure may be extracted. The chapter inherits the manuscript's source law: source must be typed by origin, modality, provenance, trust, lineage, and governance state before it can become stable structural material. But Chapter 8 adds a stricter requirement: a source is useful here not merely as evidence, but as **graph-bearing material**.

Define a source object as

`s = (a_s, k_s, o_s, p_s, m_s, mu_s, l_s, gamma_s)`

where:

- `a_s` is source address
- `k_s` is source class
- `o_s` is origin descriptor
- `p_s` is provenance record
- `m_s` is modality profile
- `mu_s` is uncertainty and source-quality profile
- `l_s` is lineage
- `gamma_s` is governance state

A source graph is then the typed graph induced from one or more sources:

`G_s = (V_s, E_s, kappa_V, kappa_E, I_s, l_s)`

where:

- `V_s` is the node family
- `E_s` is the edge family
- `kappa_V` are node kinds
- `kappa_E` are edge kinds
- `I_s` is the invariant family
- `l_s` is graph lineage

The crucial distinction is that the chapter does not treat every adjacency or co-occurrence as graph truth. Only typed, provenance-bearing, and later-replayable relations may enter the source graph.

A morphology is a stable structural shape over the source graph. It is not merely one pattern instance. It is a typed family of local graph organization that may survive normalization, rewrite, or transport.

Define a morphological signature as

`mu_G = (a_mu, chi_mu, V_mu, E_mu, iota_mu, x_mu, l_mu)`

- `chi_mu` is target scope
- `V_mu, E_mu` are local graph subfamilies
- `iota_mu` is invariant core
- `x_mu` is the exception ledger
- `l_mu` is lineage

A morphology is lawful only if it preserves a nontrivial invariant core under the rewrite family relevant to the chapter. A mere visual resemblance or one-off surface repetition is not enough.

Canonical morphology classes include:

`K_M = {Chain, Cycle, Hub, Bridge, Fork, Merge, Lattice, Shell, Spine, Mirror}`

These classes are not final equations. They are the structural atoms from which equation-like law later emerges.

### 8.1.3. Equivalence and non-equivalence classes

A major task of Chapter 8 is to determine when two source-graph regions are morphologically equivalent, merely similar, or fundamentally distinct.

Define an equivalence relation:

`~_morph`

such that for graph regions `A, B`,

## 8.5. Canonical Algorithmic Form

```python
from dataclasses import dataclass, field
from typing import Dict, List, Any

@dataclass
class SourceNode:
    address: str
    node_kind: str
    payload: Dict[str, Any]
    provenance: Dict[str, Any]
    lineage: List[str] = field(default_factory=list)

@dataclass
class SourceEdge:
    address: str
    edge_kind: str
    src: str
    dst: str
    uncertainty: Dict[str, Any]
    lineage: List[str] = field(default_factory=list)

@dataclass
class MorphSignature:
    address: str
    node_refs: List[str]
    edge_refs: List[str]
    invariant_core: Dict[str, Any]
    exception_ledger: List[str]
    lineage: List[str] = field(default_factory=list)

def build_source_graph(nodes: List[SourceNode], edges: List[SourceEdge]) -> Dict[str, Any]:
    return {
        "node_refs": [n.address for n in nodes],
        "edge_refs": [e.address for e in edges],
        "lineage": [n.address for n in nodes] + [e.address for e in edges],
    }

def extract_morph_signature(graph: Dict[str, Any]) -> MorphSignature:
    return MorphSignature(
        address="morph/auto",
        node_refs=list(graph["node_refs"]),
        edge_refs=list(graph["edge_refs"]),
        invariant_core={},
        exception_ledger=[],
        lineage=list(graph["lineage"]),
    )

def synthesize_equation_candidate(sig: MorphSignature) -> Dict[str, Any]:
    return {
        "equation_candidate_address": f"{sig.address}/eq",
        "signature_ref": sig.address,
        "requires_replay": True,
        "exception_ledger": list(sig.exception_ledger),
    }
```
