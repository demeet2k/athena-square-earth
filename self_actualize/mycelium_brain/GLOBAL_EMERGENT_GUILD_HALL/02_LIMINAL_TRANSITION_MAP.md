# Liminal Transition Map

## Current Transition

Athena is primarily in the transition:

`N+4 -> N+5`

using the liminal grammar:

`Organism -> Social`

## Why

Athena has already crossed the threshold from loose archive mass into an organism.
Evidence:

1. there is a whole-organism center
2. there is a runtime waist
3. there is a canonical cortex
4. there are family organs, rails, hubs, packets, ledgers, manifests, and restart laws

That means the `Cell -> Organism` move is largely complete enough to be real.

What is now being asked for is different:

- many agents in tandem
- shared frontier claims
- shared message boards
- shared plans
- one visible council surface
- one social coordination organ

Those are the marks of `Organism -> Social`.

## Secondary Transition

Athena is also still stabilizing late `N+3 -> N+4` behavior in weaker zones.

This shows up as:

- low-mass families that are still seed-thin
- mirror drift between precursor and live nervous systems
- packets and manifests that do not yet carry full tensor depth
- queue and ledger synchronization lag

So the true state is:

- primary transition: `N+4 -> N+5`
- residual stabilization: `N+3 -> N+4`

## Current Obstacle

The core obstacle of the current transition is:

`specialize and coordinate without fragmenting`

The system can already differentiate.
It still struggles to keep all differentiated parts socially coherent as they change in real time.

## Required Catalyst

The main catalyst is:

`shared visible coordination infrastructure with lawful hardening`

That means:

1. one guild hall
2. one message-board layer
3. one shared future-plan layer
4. one proposal zone
5. one change feed
6. one rule for promoting board matter into canonical or runtime matter

The transition is stabilizing if:

1. fewer deltas are lost between center and runtime
2. more fronts become visible before they sprawl
3. future plans are separated cleanly from canon and runtime

3. message boards multiply without coupling to queues and receipts
4. new plans appear faster than old ones are resolved
5. the social layer increases heat faster than it increases clarity
