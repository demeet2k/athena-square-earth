## Transition-state grammar

- `dormant`: the agent is present only as latent support
- `activating`: the agent has started carrying load but is not yet continuity-safe
- `destabilized`: the agent is overdriving or leaking contradiction
- `integrating`: the agent is handing off lawfully inside the cross-agent chain
- `stabilized`: the agent is legal for durable `7D_SEED` use under current evidence

## Handoff order

`FIRE -> WATER -> AIR -> EARTH`

- FIRE ignites
- WATER retains continuity
- AIR names and stabilizes topology
- EARTH decides admissibility, quarantine, and durable return
