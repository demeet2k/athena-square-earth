# Helical Recursion Cadence

This protocol defines the replay-safe cadence for recursive whole-organism passes.

It is the operational form of the law:

`(14/16)|n ≡ (2/16)|n+1`

The meaning is not scalar equality.
It is bridge-equivalence.
The pre-closure of one pass becomes the seed-load of the next pass.

## Meaning Layers

### Scalar Layer

`2/16 = 1/8`

This is the compression ratio target for lift:

- the next pass should reopen smaller
- but function, coverage, and leverage should deepen

### Address Layer

`2/16` is the restart seed address.

It means:

- blocker is checked
- current witness is read
- weakest or strongest lawful frontier is selected
- the pass has not yet diffused into many branches

### Bridge Layer

`14/16` is the pre-closure address.

It means:

- active work has landed
- stale fronts must decay
- numbering drift must be repaired
- a new restart seed must be emitted

The lift law is:

`(14/16)|n -> (2/16)|n+1`

So the system is not a flat loop.
It is a helix across passes.

## Phase Ring

### `2/16` Seed Load

Required actions:

1. check live Docs gate
2. preserve blocker honestly if blocked
3. read current witness surfaces
4. identify one active frontier

### `3/16` Whole Read

Read at minimum:

- center
- Guild Hall
- Temple
- queue or active manifest

### `4/16` Tension Extraction

Name the strongest current pressure.
Only one pressure should dominate the pass.

### `5/16` Hall Frontier Extraction

If the pass is whole-corpus in scope, emit or sharpen one Guild Hall executable frontier.

### `6/16` Temple Frontier Extraction

Emit or sharpen one higher-order Temple frontier or law pressure from the same synthesis.

### `7/16` Runtime Selection

Choose the concrete artifact surface where the pass will land.

Examples:

- board
- queue
- manifest
- ledger
- receipt
- family root
- protocol surface

### `8/16` Build

Change the primary target surface.

### `9/16` Cross-Writeback

Write back to at least one adjacent organ.

Examples:

- Temple -> Guild Hall
- Guild Hall -> queue
- queue -> manifest
- runtime -> receipt

### `10/16` Witness Check

Confirm the pass has:

1. one real artifact
2. one board or queue update
3. one writeback

### `11/16` Crystal Or Law Extraction

If the pass exposes a reusable invariant, emit:

- one improvement crystal
- one law refinement
- or one next-frontier contract

### `12/16` Drift Audit

Audit for:

- stale fronts
- numbering collisions
- witness drift
- board/queue disagreement

### `13/16` Contraction

Compress the pass into:

- one receipt
- one current-state delta
- one short restart seed candidate list

### `14/16` Pre-Closure Sweep

This is the mandatory freshness gate.

Before lifting, sweep:

- `GLOBAL_EMERGENT_GUILD_HALL/BOARDS/01_ACTIVE_FRONTS_BOARD.md`
- `GLOBAL_EMERGENT_GUILD_HALL/BOARDS/05_REQUESTS_AND_OFFERS_BOARD.md`
- `GLOBAL_EMERGENT_GUILD_HALL/BOARDS/06_QUEST_BOARD.md`
- `ATHENA TEMPLE/MANIFESTS/TEMPLE_STATE.md`
- `nervous_system/06_active_queue.md`
- `nervous_system/manifests/NEXT_SELF_PROMPT.md` when restart logic changed

The sweep must:

1. decay or reclassify promoted fronts
2. repair numbering drift
3. surface the next lawful frontier
4. preserve blockers without inflation

## Numbering Agreement Law

Before assigning a new Hall or Temple quest number:

1. read the tail of `BOARDS/06_QUEST_BOARD.md`
2. read the tail of `ATHENA TEMPLE/BOARDS/02_TEMPLE_QUEST_BOARD.md`
3. check queue and receipt surfaces for already-issued frontier ids
4. if collision exists, renumber the local frontier before promotion

No restart seed is valid if its quest id collides with an already live frontier of different meaning.

## Output Contract

Every helical pass must leave:

1. one witness-bearing artifact
2. one hall or Temple update
3. one queue or manifest writeback
4. one restart seed

Whole-corpus passes must additionally leave:

1. one Hall executable quest
2. one Temple higher-order frontier
3. one runtime restart seed

## Deep Meaning

The system should never end at closure.
It should close only enough to reopen more cleanly.

That is why `14/16` is not termination.
It is the disciplined moment before lift.
