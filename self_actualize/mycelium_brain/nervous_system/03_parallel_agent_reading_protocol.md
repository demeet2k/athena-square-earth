# Parallel Agent Reading Protocol

## Goal

Systematically break the corpus into nested reading agents so the project can be synthesized at many scales without losing structure.

## Layer 1: Family Agents

One agent per major family:

- Voynich agent
- MATH agent
- ECOSYSTEM agent
- self_actualize agent
- DEEPER CRYSTALIZATION agent
- Trading Bot agent
- NERUAL NETWORK agent
- Books/FRESH/root-manuscript agent

## Layer 2: Manuscript Agents

Inside each family, one agent per major manuscript or framework body.

Examples:

- `MYCELIUM METRO`
- `VOID_CH11`
- `ATHENA BECOME`
- `Q-SHRINK`
- `AtlasForge`
- `The Holographic Manuscript Brain`

## Layer 3: Chapter Agents

Each manuscript agent assigns one agent per chapter or equivalent section cluster.

## Layer 4: Section Agents

Each chapter agent assigns one agent per section.

## Layer 5: Atomic Node Agents

Each section agent expands into atomic nodes or addressable units.

## Contraction Order

1. atomic node -> section
2. section -> chapter
3. chapter -> manuscript
4. manuscript -> family
5. family -> corpus

## Required Outputs At Every Layer

- local summary seed
- emergent metro lines
- unresolved contradictions
- witness anchors
- next compression target

## Active Rule

No layer may collapse upward until its siblings have been compared for emergent lines.
